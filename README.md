# Strategic Tech Partnership Proposal

An interactive single-page web application presenting a SaaS partnership pitch deck with live demos and financial projections.

## Features

- **Tab-based Dashboard** - Navigate through Vision, Solution, and Business Deal sections
- **Live HACCP Demo** - Real-time updating chart demonstrating ESP32 IoT concept
- **Financial Calculator** - Interactive projections with adjustable parameters
- **Ecosystem Visualization** - Integration diagram for Toast, WP, and Custom App
- **Training/Rewards Modules** - Card-based layout showcasing platform features

## Tech Stack

- HTML5
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- [Chart.js](https://www.chartjs.org/) for data visualization
- Google Fonts (Playfair Display, Inter)
